# 綠豆實驗：ImageJ 光強度測量與校準指南

本指南說明如何使用 ImageJ 軟體來測量 LED 的相對光強度，並據此調整 ESP32-C3 的 PWM 數值，以確保實驗變因（光強度）的一致性。

## 1. 拍攝校準照片 (關鍵步驟)
在分析之前，拍攝照片的方式必須完全一致，否則數據將無意義：
*   **固定相機參數**：使用手機或相機時，必須**鎖定曝光 (AE Lock)**、**鎖定 ISO** 與 **鎖定白平衡 (WB)**。不可使用自動模式。
*   **使用擴散片**：在 LED 前方放一張**半透明白紙**或磨砂塑膠片。不要直接對著燈珠拍，這會導致感光元件過曝（呈現純白色），無法測量強度差異。
*   **環境光控制**：關閉房間內的所有其他燈光，確保照片中只有 LED 的光。
*   **固定距離**：相機與 LED 的距離在三組照片中必須完全相同。

## 2. ImageJ 操作步驟
1.  **開啟影像**：將拍攝好的照片拖入 ImageJ。
2.  **轉換為灰階**：
    *   點選選單 `Image > Type > 8-bit`。這會將彩色照片轉為 0 (黑) 到 255 (白) 的灰階數值。
3.  **選取測量區域**：
    *   使用工具列上的圓形或矩形選取工具，框住照片中「光最亮」的區域。
    *   **注意**：每一張照片（紅、紫、白）選取的框框大小和位置必須一致。
4.  **設定測量參數**：
    *   點選 `Analyze > Set Measurements...`。
    *   勾選 `Mean Gray Value` (平均灰階值) 和 `Min & Max Gray Value`。
5.  **執行測量**：
    *   按下快捷鍵 `Ctrl + M` (或點選 `Analyze > Measure`)。
    *   彈出的 `Results` 視窗中，**Mean** 欄位即代表該光的相對強度。

## 3. 如何校準程式碼
假設您測得的初步數據如下：
*   **白光組** (PWM 255,255,255)：Mean = 220
*   **紅光組** (PWM 255,0,0)：Mean = 120
*   **紫光組** (PWM 255,0,255)：Mean = 180

為了讓強度一致，您需要**降低白光與紫光的 PWM 亮度**，使其 Mean 值接近紅光組的 120：
1.  在 Serial Monitor 輸入指令調低數值（例如 `W:140,140,140`）。
2.  重新拍照並用 ImageJ 測量。
3.  重覆此過程，直到三組光的 Mean 值誤差在 ±5% 以內。

## 4. 實驗注意事項
*   **過曝檢測**：如果 ImageJ 測出的 Max 值是 255，且 Mean 值非常高，代表照片「爆掉」了。請調低相機曝光值或增加擴散片厚度後重新拍攝。
*   **光譜靈敏度**：感光元件對不同顏色的敏感度不同，雖然 ImageJ 是灰階分析，但這已是目前非專業設備下最接近科學化的測量方式。
